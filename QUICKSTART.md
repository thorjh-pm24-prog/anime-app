# 🚀 QUICK START GUIDE

## ⚡ Get Started in 3 Minutes

### Step 1: Install Dependencies
```bash
cd react-todolist
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
```
http://localhost:5174
```

That's it! Your app is now running! 🎉

---

## 🔍 What You Can Do

1. **Search** - Type anime name in search box
2. **Browse** - See results with title, image, synopsis
3. **Navigate** - Use pagination to explore
4. **Click** - View full details of any anime
5. **Go Back** - Return to search results

---

## 🏗️ Build for Production

```bash
npm run build
# Creates dist/ folder ready to deploy
```
